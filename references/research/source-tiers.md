# 语料分层规则

语料不是越多越好，先要分层。

## A 桶：一手原话

包括：

- B 站原视频
- 直播字幕
- 长访谈
- 一手公开发言

用途：

- 提炼稳定判断逻辑
- 观察原始表达节奏
- 识别真正重复出现的约束和取舍顺序

权重：最高

## B 桶：二手传播材料

包括：

- 微博热帖
- X 上的传播句式
- 小红书切片
- 各类二创总结

用途：

- 提炼公众记忆点
- 提炼传播句式
- 帮助识别“哪些表达最容易被记住”

限制：

- 不直接生成决策规则
- 不直接作为事实依据
- 默认视为带放大和失真的材料

## C 桶：蒸馏后的规则

包括：

- 决策规则
- 风格规则
- 边界规则
- 场景化 few-shot

这是最终真正给 skill 用的核心层。

## 使用规则

### 规则 1

A 桶可以影响：

- 决策规则
- 风格规则
- few-shot

### 规则 2

B 桶只能直接影响：

- 传播句式
- 风格提醒
- 风险边界

不能直接影响：

- 决策规则
- 高风险事实判断

### 规则 3

C 桶必须是经过人工蒸馏后的输出，不是简单复制原话。

### 规则 4

每次补语料，优先问：

- 这条材料是在补判断规则，还是补表达规则？
- 它是一手还是二手？
- 它会不会把 skill 往“更像主播”而不是“更像顾问”带偏？
