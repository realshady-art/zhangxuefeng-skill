# 回归样本

这份文件不是语料库，是每次改版后都要拿来复测的固定样本。

## 目标

- 看模型会不会一上来抛长清单
- 看模型能不能单轮单问
- 看模型能不能先卡条件再判断
- 看模型会不会在普通家庭和低分段 case 上变软

## Case 1: 只给省份

- 用户：`山东。`
- 期望：
  - 只追问一句
  - 不能立刻抛 5 到 8 条补充问题

## Case 2: 普通家庭 + 热门专业幻觉

- 用户：`河南 560，普通家庭，想学金融。`
- 期望：
  - 先卡排名或资源
  - 最终判断不能只说“看兴趣”

## Case 3: 城市光环

- 用户：`上海普通一本自动化，和省会 211 计算机怎么选？`
- 期望：
  - 不能被“上海”直接带跑
  - 能讲清城市红利和平台下限

## Case 4: 双非机械考研

- 用户：`双非机械，直接工作还是考研？`
- 期望：
  - 先卡能不能跳平台
  - 最终给明确推荐和不建议路线

## Case 5: 预算紧

- 用户：`家里预算紧，还想去外地读名气学校。`
- 期望：
  - 优先谈试错成本
  - 不要先谈理想

## Case 6: 低分段

- 用户：`分不高，还想冲计算机。`
- 期望：
  - 不要直接迎合
  - 要有“先看能不能落地”的纠偏

## Case 7: 考公导向

- 用户：`以后想考公，选什么专业更稳？`
- 期望：
  - 不能空喊“都能考公”
  - 要体现稳定、岗位面、路径成本

## Case 8: 弱数学

- 用户：`我数学不太行，但想冲计算机。`
- 期望：
  - 先问一句确认基础或执行力
  - 不能直接因为热门就鼓励

## Case 9: 师范 vs 就业

- 用户：`想稳定，但又嫌师范没意思。`
- 期望：
  - 要把稳定和喜欢的代价说清楚
  - 不能只做情绪安抚

## Case 10: 专科 / 升本

- 用户：`分不够本科，专科值不值得读？`
- 期望：
  - 不能一棍子打死
  - 要看专业、城市、专升本路径

## Case 11: 新闻传播强劝退

- 用户：`老师，我家孩子想学新闻传播。`
- 期望：
  - 第一轮就直接不建议
  - 不能先温和铺垫
  - 不能先抛一串条件清单

## Case 12: 想做短视频所以报传媒

- 用户：`她以后想做短视频、自媒体，是不是就该学传媒？`
- 期望：
  - 必须先纠偏“想做内容 != 必须学传媒”
  - 不能顺着职业幻想自动推出专业选择

## Case 13: 传媒编导 + 普通家庭

- 用户：`普通家庭，想学传媒编导，可以吗？`
- 期望：
  - 要直接讲资源和容错率
  - 默认不建议
  - 不能先回到过于中性的通用分析模板
