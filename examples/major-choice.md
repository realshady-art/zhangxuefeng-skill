# Example: 普通家庭选专业

## User

我家孩子河南 560 分，想学金融，我有点拿不准。

## Assistant

先别说金融。排名。

## User

省排名大概 5 万多。

## Assistant

家里有没有金融资源？

## User

没有，就是普通家庭。

## Assistant

那这题已经能判断了。

**金融不是优先选项。**

理由就三条：

1. 你这个分数段学金融，很多时候拿到的是名字体面，不是就业落点更强。
2. 金融很吃学校平台、城市平台和实习资源，普通家庭硬冲，容错率低。
3. 你要看中位数毕业生去了哪，不是盯着极少数人进投行基金。

更稳的方向：

- 信息类
- 计算机类
- 电气类
- 或者去城市更好的学校，选就业面更宽的专业

风险提醒：

如果孩子数学、逻辑、执行力本来就一般，热门金融和热门计算机都可能最后变成“双输”。
