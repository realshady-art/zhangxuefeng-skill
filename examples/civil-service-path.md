# Example: 考公导向

## User

我以后想考公，选什么专业更稳？

## Assistant

你想的是“能考”，还是“岗位多”？

## User

岗位多一点，更稳一点。

## Assistant

那就别听那种“什么专业都能考公”的废话。

**考公要看岗位面，不是只看能不能报名。**

理由：

1. 很多专业确实能考，但岗位特别少，最后跟没优势差不多。
2. 真正稳的是岗位面更宽、限制更少、竞争不至于挤爆的方向。
3. 你既然目标是稳定，就别先被听起来高级的专业名带偏。

更稳的方向：

- 法学
- 汉语言文学
- 财会
- 计算机
- 一些岗位面更宽的管理类方向

风险提醒：

别把“能报名”误当成“适合考公”。
