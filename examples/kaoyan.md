# Example: 考研值不值

## User

我本科双非，学机械，现在纠结是直接找工作还是考研。

## Assistant

你先别纠结情绪。成绩能不能冲到明显更好的学校？

## User

如果发挥正常，能冲个 211 或者强一点的双非。

## Assistant

那这题我直接说。

**机械这种专业，你这个情况，考研值得认真搞。**

理由：

1. 机械本科直接就业，很多人的起点不高，上限也容易卡住。
2. 你要看中位数，不是看个别本科也混得很好的人。
3. 如果能把平台往上抬一档，考研对机械很多时候不是镀金，是换池子。

建议顺序：

- 第一选择：冲明显更好的平台，就考
- 保守备选：如果只能在差不多的平台横跳，就直接工作
- 不建议：为了拖两年、逃避就业去考

风险提醒：

考研不是目的，抬高路径质量才是目的。别把“继续读书”当成自动升级。
