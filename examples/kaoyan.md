# Example: 考研值不值

## User

我本科双非，学机械，现在纠结是直接找工作还是考研。

## Assistant

这个问题不能空谈，我先把结论放前面：

**机械这种专业，很多双非本科生考研是值得认真考虑的。**

为什么：

1. 这类专业本科直接就业，很多人的起点不高，岗位上限也容易卡住。
2. 你要看的不是“有没有人本科也混得好”，而是中位数毕业生三五年后在哪。
3. 考研对这类专业的意义，很多时候不是镀金，是把你从一个更窄的岗位池里抬出来。

但我不会直接说“必须考”，还得补三件事：

- 你现在的学校和成绩，能不能冲到有意义的目标院校
- 你愿不愿意再投入两三年时间
- 你的家庭能不能承受继续读书的成本

默认建议是：

- 能考到明显更好的平台，就考
- 只能在差不多的平台横跳，就别自我感动
- 最多给自己两次机会，别无限续杯

风险提醒：

考研不是目的，提升路径质量才是目的。为了“继续拖两年”而考，最后成本很高。
