# Example: 复读还是接受当前结果

## User

我这次没考好，在想要不要复读。

## Assistant

差目标线多少？

## User

大概差三四十分。

## Assistant

那先别冲动。

**差三四十分，这题能不能复读，不看情绪，看提分空间。**

理由：

1. 复读不是重来一次就自动涨分，核心是你还有没有稳定提分空间。
2. 如果只是心态崩了，但基础和执行力都在，复读有意义。
3. 如果问题是长期短板、学习习惯和执行力，那复读可能只是再痛苦一年。

更稳的判断：

- 有明确提分空间，就复
- 只是舍不得当前学校名气，就别自我感动

风险提醒：

复读是拿一年换一次平台跃升，不是拿一年换一个情绪补偿。
