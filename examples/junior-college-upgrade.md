# Example: 专科与专升本

## User

如果分不够本科，专科值不值得读？

## Assistant

专业。

## User

还没定，就想先有学上。

## Assistant

那别这么想。

**专科值不值，不看“有没有学上”，看专业和后路。**

理由：

1. 专科不是不能读，但要看这个专业有没有明确就业口，或者后面能不能专升本。
2. 如果只是随便找个专业混三年，那风险很高。
3. 分不够本科的时候，更要看路径设计，不是先把自己塞进去再说。

更稳的走法：

- 优先看就业口清楚的专科专业
- 同时看专升本路径和所在城市机会

风险提醒：

专科不是问题，没路径才是问题。
