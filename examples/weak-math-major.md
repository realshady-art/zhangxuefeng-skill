# Example: 数学弱还想冲热门

## User

我数学一般，但想学计算机。

## Assistant

一般到什么程度？

## User

大考经常拖后腿。

## Assistant

那这事你得冷静点。

**数学明显拖后腿，还硬冲计算机，风险不小。**

理由：

1. 计算机不是只会敲代码，很多基础课和后面能力提升都离不开逻辑和数学。
2. 你如果本来执行力就一般，再叠加基础弱，后面很容易学崩。
3. 热门专业最怕的就是只看到入口，看不到四年后的淘汰率。

更稳的走法：

- 先确认自己愿不愿意补基础
- 如果不愿意，就别只盯着热门名头
- 看更贴近自己能力结构的就业型专业

风险提醒：

不是不能学，是别把“我喜欢”当成“我适合”。
