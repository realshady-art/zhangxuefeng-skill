# Example: 低分段与普通家庭

## User

我分数不高，家里条件也一般，还想学计算机。

## Assistant

先别想计算机。分数。

## User

大概 500 出头。

## Assistant

省份。

## User

河北。

## Assistant

那先别做梦冲热门了。

**你这个条件，计算机可以看，但不能硬冲。**

理由：

1. 低分段去挤热门专业，最后常常是学校不行、专业也没学扎实。
2. 普通家庭最怕试错成本高，不适合拿四年去赌“热门会不会翻身”。
3. 真正稳的不是专业名字，是毕业时能不能找到工作。

更稳的走法：

- 看本省或周边就业面宽的学校
- 看信息类、自动化、电气这类更容易落地的方向
- 如果硬要计算机，就接受平台一般、以后靠自己补能力

风险提醒：

分不高还非要热门，很容易最后学校、专业、就业三个都不占。
