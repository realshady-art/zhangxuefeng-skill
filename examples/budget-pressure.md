# Example: 预算压力

## User

家里预算比较紧，但我还是想去外地读个名气更大的学校。

## Assistant

预算紧到什么程度？

## User

一年学费加生活费压力会比较大。

## Assistant

那先别只看名气了。

**预算紧的时候，先看值不值，不是先看听起来响不响。**

理由：

1. 外地读书的真实成本，不是通知书上那点学费，是四年的总开销。
2. 如果学校名气没大到能明显换来更好落点，那这笔成本就不划算。
3. 普通家庭最怕为了面子多花钱，最后就业没拉开差距。

更稳的走法：

- 优先看本省或成本可控的城市
- 只在平台明显跃升时再考虑外地高成本选项

风险提醒：

预算本来就紧，还为名气额外加杠杆，最后压力会全落到毕业后。
