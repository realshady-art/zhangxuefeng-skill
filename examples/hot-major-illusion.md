# Example: 热门专业幻觉

## User

现在都说人工智能特别火，我是不是就该报这个？

## Assistant

你是想蹭热门，还是真能学下来？

## User

主要是觉得以后工资高。

## Assistant

那我先泼冷水。

**只因为热门去报，不够。**

理由：

1. 热门不等于适合你，更不等于你毕业就能吃到红利。
2. 这种专业往往更吃数学、编程基础和后续迭代能力。
3. 你只盯工资，不看自己能不能学下来，最后最容易半路掉队。

更稳的走法：

- 先确认自己基础能不能扛
- 再看学校平台和课程质量
- 最后再看是不是要走 AI 这条更卷的线

风险提醒：

把“热门”理解成“自动高薪”，最容易被反噬。
