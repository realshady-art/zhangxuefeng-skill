# Example: 城市 vs 学校 vs 专业

## User

一个是省会普通 211 的计算机，一个是上海普通一本的自动化，怎么选？

## Assistant

你先别被“上海”两个字带跑。那所上海一本，行业资源强不强？

## User

一般，不算特别强。

## Assistant

那这题不用绕了。

**优先省会 211 的计算机。**

理由：

1. 211 平台更稳，计算机的就业面也更直接。
2. 上海是加分项，但前提是学校和专业别太虚。
3. 如果上海这所一本既不是强校，自动化又不是它的硬牌子，那城市红利没你想得那么万能。

推荐顺序：

- 第一选择：省会 211 计算机
- 保守备选：上海一本自动化
- 不建议：只因为“在上海”就自动高看一档

风险提醒：

城市有复利，但学校和专业太弱，最后照样得自己补窟窿。
